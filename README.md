📌 Descrição do Projeto

Aplicação web de conversão de moedas em tempo real, desenvolvida para simular o funcionamento de um conversor financeiro moderno. O sistema permite que o usuário insira um valor em reais (BRL) e visualize instantaneamente a conversão para diferentes moedas internacionais.

A aplicação utiliza dados atualizados de uma API externa, garantindo maior precisão nas conversões e proporcionando uma experiência próxima de sistemas reais utilizados no mercado financeiro.

🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos essenciais do desenvolvimento web, como consumo de APIs, manipulação do DOM e lógica de programação aplicada a cenários reais.

Além disso, busca simular uma aplicação útil do dia a dia, demonstrando na prática como sistemas financeiros simples funcionam.
